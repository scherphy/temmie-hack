###requirement:

* keyboard maestro

####import script

* Clone this repository
* Launch keyboard maestro
* Then, "File"->"Import Macros..."->"temmie-hack.kmmacros"
* Switch to Undertale window
* And, go to Temmie Shop.(You must emptify inventory)
* Type ⌥+⌘+t (focus cursor "buy")
